---

JSONViewer AI provides essential features for working with JSON efficiently.

## JSON formatting
Automatically formats raw JSON into a clean and readable structure.

## Validation
Detects and highlights syntax errors and invalid JSON.

## Expand and collapse
Navigate large JSON files by expanding or collapsing objects.

## Large file support
Handles large JSON payloads smoothly without performance issues.

## Privacy-friendly
All JSON is processed locally in the browser and is not stored.
