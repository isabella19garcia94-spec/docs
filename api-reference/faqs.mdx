## Is JSONViewer AI free to use?
Yes, the core features are available for free.

## Do I need to create an account?
No account or login is required to use the tool.

## Is my JSON data stored?
No. JSON data is processed locally and not saved.

## Can it handle large JSON files?
Yes, it is optimized to handle large payloads.

## Does it support invalid JSON?
Yes, syntax errors are highlighted to help identify issues.
